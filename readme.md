#  Lembrete de Hidratação

Aplicação web simples para ajudar a manter a rotina de hidratação diária.

## Funcionalidades

* Controle do consumo de água.
* Meta diária de 2000 ml.
* Barra de progresso visual.
* Cronômetro para lembretes periódicos.
* Notificações do navegador.
* Alarme sonoro ao final do tempo.
* Tema claro e escuro.

## Tecnologias

* HTML
* CSS
* JavaScript

## Como usar

1. Escolha o intervalo do lembrete.
2. Clique em **Iniciar Cronômetro**.
3. Registre a quantidade de água consumida.
4. Receba notificações para lembrar de se hidratar.

## Objetivo

Incentivar o consumo regular de água ao longo do dia por meio de lembretes automáticos e acompanhamento da meta de hidratação.
